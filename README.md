# whalesong

Peer to peer chat system with etcd in a Docker container.

Theme song: [http://upload.wikimedia.org/wikipedia/commons/1/13/Humpbackwhale2.ogg]

## Etcd Structure

```
/whalesong/
  rooms/
    room1/
      caption
      1
      2
      3
      ...
    room2/
      caption
      1
      2
      3
```
